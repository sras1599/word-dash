Use these examples as patterns for requests that should invoke `prototype-ui`.

- "Use $prototype-ui to prototype 4 selected-card styles for `frontend/src/components/Card/Card.tsx`; make one crisp, one playful, one minimal, and one animated."
- "Use $prototype-ui with the prompt in `plans/card-selection-variants.md` and create Storybook experiments only."
- "Use $prototype-ui to explore hover, selected, disabled, and focus states for the player hand cards. Keep dimensions stable and include subtle motion."
- "Use $prototype-ui to create 3 mobile-friendly lobby button variants with different emphasis levels."
